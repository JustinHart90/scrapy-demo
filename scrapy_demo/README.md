#Scrapy Demo

##Steps:
1. Install Scrapy
  * `pip install scrapy==1.0.3`

1. Create a virtual envirnoment
  * `virtualenv env`

1. Start new project
  * `scrapy startproject <project name>`

1. Cd into project
  * `cd <project name>`

1. Run spider generator
  * `scrapy genspider <spider name> <spider url>`
